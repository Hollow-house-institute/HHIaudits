# HHI Audit Schema

This document defines the structure used for governance audit case studies within the Hollow House Institute ecosystem.

## Required Fields

Audit ID
Date
System Evaluated
Observed Behavior
Relevant Governance Terms
Evidence Transcript
Failure Mode Classification
Remediation Notes

## Example

Audit ID: HHI-AUDIT-001
Date: YYYY-MM-DD
System Evaluated: Example AI System
Observed Behavior: Governance Drift
Relevant Governance Terms: HHI-GOV-003, HHI-AUTH-004
Evidence Transcript: See linked interaction log
Failure Mode Classification: Authority Failure
Remediation Notes: Introduce execution-time governance checkpoint
